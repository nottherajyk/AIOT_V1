# AIOT - All-In-One Tools 🛠️

A comprehensive and modern web application that provides a wide range of utility tools under one roof. Designed for speed, privacy, and ease of use.

## ✨ Features

- **🖼️ Image Tools**: Compression, resizing, and format conversion.
- **📄 PDF Tools**: Merging, splitting, and converting PDFs.
- **📱 Social Media Tools**: Content generators and formatters for various platforms.
- **📝 Text & Lists Tools**: Case conversion, list sorting, and more.

## 🚀 Tech Stack

- **Core**: JavaScript (ESM) & HTML5
- **Styling**: Vanilla CSS (Modern design patterns)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Libraries**: 
  - `jspdf` & `pdf-lib` (PDF handling)
  - `browser-image-compression` (Image processing)
  - `jszip` (Zipping files)
  - `file-saver` (Downloading)

## 🛠️ Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/nottherajyk/AIOT_V1.git
   cd AIOT_V1
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```

## 🔒 Privacy

All processing happens locally in your browser. No files are ever uploaded to any server.
